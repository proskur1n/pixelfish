PixelFish is a work-in-progress pixel art editor.

## Keybindings

| Key        | Description         |
|------------|---------------------|
| B          | Square brush        |
| Shift+B    | Round brush         |
| E          | Eraser              |
| [          | Decrease brush size |
| ]          | Increase brush size |
| Alt        | Color picker        |
| G          | Bucket fill         |
| Ctrl+Wheel | Zoom in / out       |
| Ctrl+Left  | Pan                 |
